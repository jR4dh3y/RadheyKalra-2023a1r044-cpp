# File handling 
In cpp , File handling, is a process of permannently storing data in a file.

to handel files in cpp we use `<fstream>` header file.

#### operation performed on files
1. Create a file
2. Open a file
3. Read from a file
4. Write to a file
5. Close a file

#### File modes
1. open         
2. write        by `ofstreem`
3. read         by `ifstreem`
4. append       
5. read/write   by `fstream`
